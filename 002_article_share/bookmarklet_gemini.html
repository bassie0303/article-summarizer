🤖 記事シェアアプリ — ブックマークレット版 (Gemini)

Google Gemini を使ったSNSシェアツール — ブラウザ拡張不要

① ブックマークレットをインストール

下のボタンを ブックマークバーにドラッグ してください。

🤖 記事シェア(Gemini)

Chrome は ⌘+Shift+B (Mac) / Ctrl+Shift+B (Win) でブックマークバーを表示できます。

手動インストール（Android・同期できない場合はこちら）

① 「コードをコピー」を押す
② ブックマークバーを右クリック →「ページを追加…」
③ 名前: 🤖 記事シェア(Gemini)　URL欄: コードを貼り付け → 保存 

生成中...
このページで動作テスト

外部JSを読み込んでパネルが開きます。

💡 なぜ短くなった？
パネルのロジックを bookmarklet_gemini.js として GitHub に置き、ブックマークには「そのJSを読み込む小さなコード」だけを登録する方式に変更しました。
ブックマークURL は約 — 文字で、Chrome 同期・Android でも問題なく動作します。

② 使い方

  1. 要約したい記事をブラウザで開く
  2. ブックマークバーの「🤖 記事シェア(Gemini)」をクリック
  3. 初回のみ Google AI Studio の API キーを入力（次回から自動入力）
  4. 「✨ 要約を生成」をクリック
  5. X・Facebook・Threads 向けテキストをコピーまたは直接シェア
💡 もう一度クリックするとパネルが閉じます。

🔑 API キーの取得

Google AI Studio から無料で取得できます。キーは AIzaSy... から始まります。
キーはブラウザの localStorage にのみ保存され、第三者サーバーには送信されません。
