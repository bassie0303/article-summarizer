① ブックマークレットをインストール

下のボタンを ブックマークバーにドラッグ してください。

📢 記事シェア（要約版）

ブックマークバーが表示されていない場合：Chrome ⌘+Shift+B / Firefox 表示 → ツールバー → ブックマークツールバー

ドラッグがうまくいかない場合 → 手動インストール（確実）

① 下の「コードをコピー」を押す
② ブックマークバーを右クリック →「ページを追加…」
③ 名前: 📢 記事シェア（要約版）　URL欄: コピーしたコードを貼り付け → 保存 

生成中...
このページで動作テスト

インストール前に動作確認できます。

② 使い方

  1. 要約したい記事をブラウザで開く
  2. ブックマークバーの「📢 記事シェア（要約版）」をクリック
  3. 初回のみ Anthropic API キーを入力（次回から自動入力）
  4. 「✨ 要約を生成」をクリック
  5. X・Facebook・Threads 向けテキストをコピーまたは直接シェア
💡 もう一度クリックするとパネルが閉じます。

⚠️ セキュリティ

API キーはブラウザの localStorage にのみ保存されます。Anthropic API に直接リクエストが送信されます。
API キー未取得の場合は console.anthropic.com から取得できます。
