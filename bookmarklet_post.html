外部JSを読み込んでパネルが開きます。

💡 なぜ短くなった？
パネルのロジックを bookmarklet_post.js として GitHub に置き、ブックマークには「そのJSを読み込む小さなコード」だけを登録する方式に変更しました。
ブックマークURL は約 — 文字で、Chrome 同期・Android でも問題なく動作します。

⚠️ GitHub へのプッシュが必要です

ブックマークレットは bookmarklet_post.js を CDN から読み込みます。
リポジトリに push する前は動作しません。

  1. bookmarklet_post.js を article-summarizer リポジトリの main ブランチ に push する
  2. 数分後に jsDelivr CDN に反映される
  3. インストール済みのブックマークレットはそのまま使える（更新時もブックマーク変更不要）
git add bookmarklet_post.js
git commit -m "Add external bookmarklet script"
git push origin main
⚠️ CDN のキャッシュ反映に最大 24 時間かかる場合があります。
すぐ試したい場合は「このページで動作テスト」ボタンを使ってください（ローカル実行）。

② 使い方

  1. 投稿したい記事をブラウザで開く
  2. 引用したい部分のテキストをマウスで選択（ドラッグ）する
  3. ブックマークバーの「✍️ 記事シェア（投稿版）」をクリック
  4. 引用欄に選択テキストが自動入力される（編集可）
  5. コメントを入力する
  6. 各SNSのテキストをコピーまたは直接シェア
💡 文字数はリアルタイムで計算されます。X・Threadsは制限内に自動的に収まります。
💡 引用もコメントも両方空欄でもURL付きでシェアできます。

📋 投稿フォーマット

コメントと引用を入力すると、以下の形式で自動整形されます：
[コメント]
> [引用テキスト（複数行は各行に>）]
[記事タイトル（ThreadsとFacebookのみ）]
[URL]
#tag1 #tag2
